C - Super Ryuma

原始题面

Problem Statement

There is an infinite two-dimensional grid, and we have a piece called Super Ryuma at square (r1,c1)(r_1,c_1)(r1​,c1​). (Ryu means dragon and Ma means horse.) In one move, the piece can go to one of the squares shown below:

More formally, when Super Ryuma is at square (a,b)(a,b)(a,b), it can go to square (c,d)(c,d)(c,d) such that at least one of the following holds:

  • a+b=c+da+b=c+da+b=c+d
  • a−b=c−da-b=c-da−b=c−d
  • ∣a−c∣+∣b−d∣≤3|a-c|+|b-d|≤3∣a−c∣+∣b−d∣≤3

Find the minimum number of moves needed for the piece to reach (r2,c2)(r_2,c_2)(r2​,c2​) from (r1,c1)(r_1,c_1)(r1​,c1​).

Constraints

  • All values in input are integers.
  • 1≤r1,c1,r2,c2≤1091≤r_1,c_1,r_2,c_2≤10^91≤r1​,c1​,r2​,c2​≤109

Input

Input is given from Standard Input in the following format:

r1c1r2c2\begin{matrix} r_1&c_1\\ r_2&c_2 \end{matrix}r1​r2​​c1​c2​​

Output

Print the minimum number of moves needed for Super Ryuma to reach (r2,c2)(r_2,c_2)(r2​,c2​) from (r1,c1)(r_1,c_1)(r1​,c1​).

Sample Output 1

Sample Output 2

E - Third Avenue

原始题面

Problem Statement

There is a town represented as a two-dimensional grid with HHH horizontal rows and WWW vertical columns.

A character ai,ja_{i,j}ai,j​ describes the square at the iii-th row from the top and jjj-th column from the left. Here, ai,ja_{i,j}ai,j​ is one of the following: S, G, ., #, a, …, and z.

# represents a square that cannot be entered, and a, …, z represent squares with teleporters.

Takahashi is initially at the square represented as S. In each second, he will make one of the following moves:

  • Go to a non-# square that is horizontally or vertically adjacent to his current position.
  • Choose a square with the same character as that of his current position, and teleport to that square. He can only use this move when he is at a square represented as a, …, or z.

Find the shortest time Takahashi needs to reach the square represented as G from the one represented as S.
If the destination is unreachable, report -1 instead.

Constraints

  • 1≤H,W≤20001≤H,W≤20001≤H,W≤2000
  • ai,ja_{i,j}ai,j​ is S, G, ., #, or a lowercase English letter.
  • There is exactly one square represented as S and one square represented as G.

Input

Input is given from Standard Input in the following format:

HWa1,1...a1,W⋮aH,1...aH,W\begin{matrix} H&W\\ a_{1,1}&...&a_{1,W}\\ \vdots\\ a_{H,1}&...&a_{H,W} \end{matrix}Ha1,1​⋮aH,1​​W......​a1,W​aH,W​​

Output

Print the shortest time Takahashi needs to reach the square represented as G from the one represented as S.
If the destination is unreachable from the initial position, print -1 instead.

Sample Input 1

Sample Output 1

Let (i,j)(i,j)(i,j) denote the square at the iii-th row from the top and jjj-th column from the left.

Initially, Takahashi is at (1,1)(1,1)(1,1). One way to reach (2,5)(2,5)(2,5) in four seconds is:

  • go from (1,1)(1,1)(1,1) to (2,1)(2,1)(2,1)
  • teleport from (2,1)(2,1)(2,1) to (2,3)(2,3)(2,3), which is also an a square;
  • go from (2,3)(2,3)(2,3) to (2,4)(2,4)(2,4);
  • go from (2,4)(2,4)(2,4) to (2,5)(2,5)(2,5).

F - Programming Contest

原始题面

Problem Statement

Takahashi will participate in a programming contest, which lasts for TTT minutes and presents NNN problems.

With his extrasensory perception, he already knows that it will take Ai minutes to solve the i-th problem.

He will choose zero or more problems to solve from the N problems so that it takes him no longer than TTT minutes in total to solve them.

Find the longest possible time it takes him to solve his choice of problems.

Constraints

  • All values in input are integers.
  • 1≤N≤401≤N≤401≤N≤40
  • 1≤T≤1091≤T≤1091≤T≤109
  • 1≤Ai≤1091≤A_i≤1091≤Ai​≤109

Input

Input is given from Standard Input in the following format:

NTA1...AN\begin{matrix} N&T\\ A_1&...&A_N \end{matrix}NA1​​T...​AN​​

Output

Print the answer as an integer.

Sample Input 1

Sample Output 1

If he chooses the 111-st, 222-nd, 333-rd, and 444-th problems, it takes him 2+3+5+7=172+3+5+7=172+3+5+7=17 minutes in total to solve them, which is the longest possible time not exceeding T=17T=17T=17 minutes.

Sample Input 2

Sample Output 2

It is optimal to solve all the problems.

Sample Input 3

Sample Output 3

He cannot solve any of the problems.

Sample Input 4

Sample Output 4

If he chooses the 222-nd, 333-rd, and 777-th problems, it takes him 273555143273555143273555143 minutes in total to solve them.

解题思路

直接查肯定T, 折半搜索可以

将数据分成两部分, 一部分二进制枚举所有可能结果并排序; 另一部分二进制枚举, 二分查找前一部分结果中满足 “两部分结果之和<T<T<T” 的最大结果
