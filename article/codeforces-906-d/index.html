题解 - [CodeForces 906 D] Power Tower

发表于 2020-12-18 更新于 2021-04-25 分类于 程序设计 ， 题解

题目链接

原始题面

time limit per test: 4.5 seconds
memory limit per test: 256 megabytes
input: standard input
output: standard output

Description

Priests of the Quetzalcoatl cult want to build a tower to represent a power of their god. Tower is usually made of power-charged rocks. It is built with the help of rare magic by levitating the current top of tower and adding rocks at its bottom. If top, which is built from k−1k-1k−1 rocks, possesses power ppp and we want to add the rock charged with power wkw_kwk​ then value of power of a new tower will be {wk}p\{w_k\}^p{wk​}p

Rocks are added from the last to the first. That is for sequence w1,...,wmw_1, ..., w_mw1​,...,wm​ value of power will be

w1w2⋅⋅⋅wmw_1^{w_2^{\cdot^{\cdot^{\cdot^{w_m}}}}}w1w2⋅⋅⋅wm​​​

After tower is built, its power may be extremely large. But still priests want to get some information about it, namely they want to know a number called cumulative power which is the true value of power taken modulo mmm. Priests have nnn rocks numbered from 111 to nnn. They ask you to calculate which value of cumulative power will the tower possess if they will build it from rocks numbered l,l+1,...,rl, l + 1, ..., rl,l+1,...,r

Input

First line of input contains two integers nnn (1≤n≤1051 ≤ n ≤ 10^51≤n≤105) and mmm (1≤m≤1091 ≤ m ≤ 10^91≤m≤109)

Second line of input contains nnn integers wkw_kwk​ (1≤wk≤1091 ≤ w_k ≤ 10^91≤wk​≤109) which is the power of rocks that priests have

Third line of input contains single integer qqq (1≤q≤1051 ≤ q ≤ 10^51≤q≤105) which is amount of queries from priests to you

kth of next qqq lines contains two integers lkl_klk​ and rkr_krk​ (1≤lk≤rk≤n1 ≤ l_k ≤ r_k ≤ n1≤lk​≤rk​≤n)

Output

Output qqq integers. k-th of them must be the amount of cumulative power the tower will have if is built from rocks lk,lk+1,...,rkl_k, l_{k + 1}, ..., r_klk​,lk+1​,...,rk​

wlwl+1⋅⋅⋅wr mod mw_l^{w_{l+1}^{\cdot^{\cdot^{\cdot^{w_r}}}}}\bmod mwlwl+1⋅⋅⋅wr​​​modm

由扩展Euler定理

递归求解即可
