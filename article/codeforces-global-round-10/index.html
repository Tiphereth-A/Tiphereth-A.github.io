A - Omkar and Password

原始题面

Lord Omkar has permitted you to enter the Holy Church of Omkar! To test your worthiness, Omkar gives you a password which you must interpret!

A password is an array aaa of nnn positive integers. You apply the following operation to the array: pick any two adjacent numbers that are not equal to each other and replace them with their sum. Formally, choose an index iii such that 1≤i<n1≤i<n1≤i<n and ai≠ai+1a_i≠a_i+1ai​​=ai​+1, delete both aia_iai​ and ai+1a_i+1ai​+1 from the array and put ai+ai+1a_i+a_i+1ai​+ai​+1 in their place.

For example, for array [7,4,3,7][7,4,3,7][7,4,3,7] you can choose i=2i=2i=2 and the array will become [7,4+3,7]=[7,7,7][7,4+3,7]=[7,7,7][7,4+3,7]=[7,7,7]. Note that in this array you can’t apply this operation anymore.

Notice that one operation will decrease the size of the password by 111. What is the shortest possible length of the password after some number (possibly 000) of operations?

Input

Each test contains multiple test cases. The first line contains the number of test cases ttt(1≤t≤1001≤t≤1001≤t≤100). Description of the test cases follows.

The first line of each test case contains an integer nnn(1≤n≤2⋅1051≤n≤2⋅10^51≤n≤2⋅105) - the length of the password.

The second line of each test case contains nnn integers a1,a2,…,ana_1,a_2,…,a_na1​,a2​,…,an​ (1≤ai≤1091≤a_i≤10^91≤ai​≤109) - the initial contents of your password.

The sum of nnn over all test cases will not exceed 2⋅1052⋅10^52⋅105.

Output

For each password, print one integer: the shortest possible length of the password after some number of operations.

Example

Input
2
4
2 1 3 1
2
420 420
Output
1
2

Note

In the first test case, you can do the following to achieve a length of 111:

Pick i=2i=2i=2 to get [2,4,1][2,4,1][2,4,1]

Pick i=1i=1i=1 to get [6,1][6,1][6,1]

Pick i=1i=1i=1 to get [7][7][7]

In the second test case, you can’t perform any operations because there is no valid iii that satisfies the requirements mentioned above.

题意简述

给出一组正整数aaa, 你可以选取其中相邻的两个不同的数, 并将靠左边的数替换为这两个数的和, 同时删除靠右边的数, 问对该组数aaa持续进行该操作直到无法继续之后, 得到数组的最小长度

解题思路

如果这组数全部相等, 此时一定无法进行操作, 答案即为这组数的长度

否则, 这组数中一定存在最大值, 且一定存在一组相邻且不相等的数, 其中一个为最大值

对这两个数进行一次操作后, 我们得到的新数组中便存在唯一的最大值, 此时对它和与它相邻的数不断重复操作, 则最大值会越来越大, 且最后只剩下一个数

B - Omkar and Infinity Clock

原始题面

Being stuck at home, Ray became extremely bored. To pass time, he asks Lord Omkar to use his time bending power: Infinity Clock! However, Lord Omkar will only listen to mortals who can solve the following problem:

You are given an array aaa of nnn integers. You are also given an integer kkk. Lord Omkar wants you to do kkk operations with this array.

Define one operation as the following:

  1. Set ddd to be the maximum value of your array.
  2. For every iii from 111 to nnn, replace aia_iai​ with d−aid-a_id−ai​.

The goal is to predict the contents in the array after kkk

operations. Please help Ray determine what the final sequence will look like!

Input

Each test contains multiple test cases. The first line contains the number of cases ttt(1≤t≤1001≤t≤1001≤t≤100). Description of the test cases follows.

The first line of each test case contains two integers nnn
and kkk (1≤n≤2⋅105,1≤k≤10181≤n≤2⋅10^5,1≤k≤10^{18}1≤n≤2⋅105,1≤k≤1018) – the length of your array and the number of operations to perform.

The second line of each test case contains nnn integers a1,a2,...,ana_1,a_2,...,a_na1​,a2​,...,an​ (−109≤ai≤109-10^9≤a_i≤10^9−109≤ai​≤109) – the initial contents of your array.

It is guaranteed that the sum of nnn over all test cases does not exceed 2⋅1052⋅10^52⋅105.

Output

For each case, print the final version of array aaa
after kkk operations described above.

Example

Input
3
2 1
-199 192
5 19
5 -1 4 2 0
1 2
69
Output
391 0
0 6 1 3 5
0

Note

In the first test case the array changes as follows:

  • Initially, the array is [-199,192]. d=192d=192d=192.
  • After the operation, the array becomes [d−(−199),d−192]=[391,0][d-(-199),d-192]=[391,0][d−(−199),d−192]=[391,0].

题意简述

给出一组整数aaa, 定义操作如下:

  • 设aaa中的最大值为ddd, 将aaa中所有元素aia_iai​替换为d−aid-a_id−ai​

问经过kkk次操作之后的数组aaa是什么样的

解题思路

看到kkk的范围就知道这个变换肯定有循环节

我们可以这样做

  • 如果aaa中全为非负数, 我们不难发现其循环节长度就是222
  • 如果aaa中存在负数, 则经过一次变换后就会变为非负数

C - Omkar and Waterslide

原始题面

Omkar is building a waterslide in his water park, and he needs your help to ensure that he does it as efficiently as possible.

Omkar currently has nnn supports arranged in a line, the iii-th of which has height aia_iai​. Omkar wants to build his waterslide from the right to the left, so his supports must be nondecreasing in height in order to support the waterslide. In 111 operation, Omkar can do the following: take any contiguous subsegment of supports which is nondecreasing by heights and add 111 to each of their heights.

Help Omkar find the minimum number of operations he needs to perform to make his supports able to support his waterslide!

An array bbb is a subsegment of an array ccc if bbb can be obtained from ccc by deletion of several (possibly zero or all) elements from the beginning and several (possibly zero or all) elements from the end.

An array b1,b2,…,bnb_1,b_2,…,b_nb1​,b2​,…,bn​ is called nondecreasing if bi≤bi+1b_i≤b_{i+1}bi​≤bi+1​ for every iii from 111 to n−1n-1n−1.

Input

Each test contains multiple test cases. The first line contains the number of test cases ttt
(1≤t≤1001≤t≤1001≤t≤100). Description of the test cases follows.

The first line of each test case contains an integer nnn
(1≤n≤2⋅1051≤n≤2⋅10^51≤n≤2⋅105) - the number of supports Omkar has.

The second line of each test case contains nnn
integers a1,a2,...,ana_1,a_2,...,a_na1​,a2​,...,an​ (0≤ai≤1090≤ai≤10^90≤ai≤109) - the heights of the supports.

It is guaranteed that the sum of nnn over all test cases does not exceed 2⋅1052⋅10^52⋅105.

Output

For each test case, output a single integer - the minimum number of operations Omkar needs to perform to make his supports able to support his waterslide.

Example

Input
3
4
5 3 2 5
5
1 2 3 5 3
3
1 1 1
Output
3
2
0

Note

The subarray with which Omkar performs the operation is bolded.

In the first test case:

  • First operation:

    [5,3,2,5]→[5,3,3,5][5,3,\bold{2},5]→[5,3,\bold{3},5][5,3,2,5]→[5,3,3,5]

  • Second operation:

    [5,3,3,5]→[5,4,4,5][5,\bold{3,3},5]→[5,\bold{4,4},5][5,3,3,5]→[5,4,4,5]

  • Third operation:

    [5,4,4,5]→[5,5,5,5][5,\bold{4,4},5]→[5,\bold{5,5},5][5,4,4,5]→[5,5,5,5]

In the third test case, the array is already nondecreasing, so Omkar does 000 operations.

题意简述

给出一组非负整数aaa, 给出一组数aaa, 定义操作如下:

  • 选择其中任何一段不下降子区间并将其上所有数+1+1+1

我们可以使用最少的操作来将该数列变为不下降的数列, 输出这个不下降的数列

    直接输出

  • 当aaa是整体是非上升的时候, 如

    直接填平就好

  • 其他情况, 如

    我们可以按如下步骤化为上一种情况

    1. 填平

      得
    2. 填平

D - Omkar and Bed Wars

原始题面

Omkar is playing his favorite pixelated video game, Bed Wars! In Bed Wars, there are nnn players arranged in a circle, so that for all jjj such that 2≤j≤n2≤j≤n2≤j≤n, player j−1j-1j−1 is to the left of the player jjj, and player jjj is to the right of player j−1j-1j−1. Additionally, player nnn is to the left of player 1, and player 111 is to the right of player nnn.

Currently, each player is attacking either the player to their left or the player to their right. This means that each player is currently being attacked by either 000, 111, or 222 other players. A key element of Bed Wars strategy is that if a player is being attacked by exactly 111 other player, then they should logically attack that player in response. If instead a player is being attacked by 000 or 222 other players, then Bed Wars strategy says that the player can logically attack either of the adjacent players.

Unfortunately, it might be that some players in this game are not following Bed Wars strategy correctly. Omkar is aware of whom each player is currently attacking, and he can talk to any amount of the nnn

players in the game to make them instead attack another player - i. e. if they are currently attacking the player to their left, Omkar can convince them to instead attack the player to their right; if they are currently attacking the player to their right, Omkar can convince them to instead attack the player to their left.

Omkar would like all players to be acting logically. Calculate the minimum amount of players that Omkar needs to talk to so that after all players he talked to (if any) have changed which player they are attacking, all players are acting logically according to Bed Wars strategy.

Input

Each test contains multiple test cases. The first line contains the number of test cases ttt(1≤t≤1041≤t≤1041≤t≤104). The descriptions of the test cases follows.

The first line of each test case contains one integer nnn(3≤n≤2⋅1053≤n≤2⋅10^53≤n≤2⋅105) - the amount of players (and therefore beds) in this game of Bed Wars.

The second line of each test case contains a string sss
of length nnn. The jjj-th character of sss is equal to L if the jjj-th player is attacking the player to their left, and R if the jjj-th player is attacking the player to their right.

It is guaranteed that the sum of nnn over all test cases does not exceed 2⋅1052⋅10^52⋅105.

Output

For each test case, output one integer: the minimum number of players Omkar needs to talk to to make it so that all players are acting logically according to Bed Wars strategy.

It can be proven that it is always possible for Omkar to achieve this under the given constraints.

Example

Input
5
4
RLRL
6
LRRRRL
8
RLLRRRLL
12
LLLLRRLRRRLL
5
RRRRR
Output
0
1
1
3
2

Note

In the first test case, players 111 and 222 are attacking each other, and players 333 and 444 are attacking each other. Each player is being attacked by exactly 111 other player, and each player is attacking the player that is attacking them, so all players are already being logical according to Bed Wars strategy and Omkar does not need to talk to any of them, making the answer 000.

In the second test case, not every player acts logically: for example, player 333 is attacked only by player 222, but doesn’t attack him in response. Omkar can talk to player 333 to convert the attack arrangement to LRLRRL, in which you can see that all players are being logical according to Bed Wars strategy, making the answer 111.

题意简述

nnn个人站成一圈, 编号0,1,2,...,n−10,1,2,...,n-10,1,2,...,n−1, 第(i+1) mod n(i+1)\bmod n(i+1)modn个人在第iii个人右边, 他们会攻击某个相邻的人, 定义操作如下:

  • 选择一个人, 调转其攻击方向

经过若干次操作后, 这nnn个人满足

  • 如果他被111个人攻击, 则他会攻击攻击他的人
  • 如果他被000个或222个人攻击, 则他可以攻击任意相邻的人

给出这nnn个人的初始攻击方向, 问最小操作数

解题思路

当我看到这题的tag有 dp 和 string suffix structures 时我是震惊的

其实我们只需让环中没有333个连续的L或R即可

我们这样操作

  • 若存在444个及以上连续的的L或R, 将第333个人的方向调转
  • 若存在333个连续的的L或R, 将第222个人的方向调转

容易证明这样操作即是最优操作

F - Omkar and Landslide

原始题面

Omkar is standing at the foot of Celeste mountain. The summit is nnn meters away from him, and he can see all of the mountains up to the summit, so for all 1≤j≤n1≤j≤n1≤j≤n he knows that the height of the mountain at the point jjj meters away from himself is hjh_jhj​ meters. It turns out that for all jjj satisfying 1≤j≤n−1,hj<hj+11≤j≤n-1, h_j<h_{j+1}1≤j≤n−1,hj​<hj+1​(meaning that heights are strictly increasing).

Suddenly, a landslide occurs! While the landslide is occurring, the following occurs: every minute, if hj+2≤hj+1h_j+2≤h_{j+1}hj​+2≤hj+1​
, then one square meter of dirt will slide from position j+1j+1j+1 to position jjj, so that hj+1h_{j+1}hj+1​ is decreased by 111 and hjh_jhj​ is increased by 111. These changes occur simultaneously, so for example, if hj+2≤hj+1h_j+2≤h_{j+1}hj​+2≤hj+1​ and hj+1+2≤hj+2h_{j+1}+2≤h_j+2hj+1​+2≤hj​+2 for some jjj, then hjh_jhj​ will be increased by 111, hj+2h_j+2hj​+2 will be decreased by 111, and hj+1h_{j+1}hj+1​ will be both increased and decreased by 111, meaning that in effect hj+1h_{j+1}hj+1​ is unchanged during that minute.

The landslide ends when there is no jjj such that hj+2≤hj+1h_j+2≤h_{j+1}hj​+2≤hj+1​. Help Omkar figure out what the values of h1,…,hnh_1,…,h_nh1​,…,hn​

will be after the landslide ends. It can be proven that under the given constraints, the landslide will always end in finitely many minutes.

Note that because of the large amount of input, it is recommended that your code uses fast IO.

Input

The first line contains a single integer nnn
(1≤n≤1061≤n≤10^61≤n≤106).

The second line contains nnn integers h1,h2,…,hnh_1,h_2,…,h_nh1​,h2​,…,hn​ satisfying 0≤h1<h2<⋯<hn≤10120≤h_1<h_2<⋯<h_n≤10^{12}0≤h1​<h2​<⋯<hn​≤1012 - the heights.

Output

Output nnn integers, where the jjj-th integer is the value of hjh_jhj​ after the landslide has stopped.

Example

Input
4
2 6 7 8
Output
5 5 6 7

Note

Initially, the mountain has heights 2,6,7,82,6,7,82,6,7,8.

In the first minute, we have 2+2≤62+2≤62+2≤6, so 222 increases to 333 and 666 decreases to 555, leaving 3,5,7,83,5,7,83,5,7,8.

In the second minute, we have 3+2≤53+2≤53+2≤5
and 5+2≤75+2≤75+2≤7, so 333 increases to 444, 555 is unchanged, and 777 decreases to 666, leaving 4,5,6,84,5,6,84,5,6,8.

In the third minute, we have 6+2≤86+2≤86+2≤8, so 666 increases to 777 and 888 decreases to 777, leaving 4,5,7,74,5,7,74,5,7,7.

In the fourth minute, we have 5+2≤75+2≤75+2≤7, so 555 increases to 666 and 777 decreases to 666, leaving 4,6,6,74,6,6,74,6,6,7.

In the fifth minute, we have 4+2≤64+2≤64+2≤6, so 444 increases to 555 and 666 decreases to 555, leaving 5,5,6,75,5,6,75,5,6,7.

In the sixth minute, nothing else can change so the landslide stops and our answer is 5,5,6,75,5,6,75,5,6,7.

题意简述

给出一组严格递增的数aaa, 如果ai+2⩽ai+1a_i+2\leqslant a_{i+1}ai​+2⩽ai+1​, 则将ai+1a_{i+1}ai+1​减111, aia_iai​加111, 输出不能进行上述操作后的aaa

解题思路

比赛花1h写了堆shit还wa了, 菜 我 菜

设S=∑i=1naiS=\sum_{i=1}^n a_iS=∑i=1n​ai​, 我们可以证明:

  • 最后结果只与SSS和nnn有关, 或者说, 对于相同的SSS和nnn, 最后结果是相同的
  • 最后结果只有一对相邻数的值相等, 其余均满足: 后一项=前一项+1+1+1

要注意这些性质不具有平凡性, 正是由于初始的aaa严格递增, 才有了这些性质

由这些, 我们便可以这样生成结果:

  • 首先令ai=ia_i=iai​=i, 这样还有S−n(n+1)2S-{n(n+1)\over2}S−2n(n+1)​个数没有分配
  • 接着把每个aia_iai​都加上⌊S−n(n+1)2n⌋\lfloor{S-{n(n+1)\over2}\over n}\rfloor⌊nS−2n(n+1)​​⌋, 这样还有(S−n(n+1)2) mod n(S-{n(n+1)\over2})\bmod n(S−2n(n+1)​)modn个数没有分配
  • 最后便把这(S−n(n+1)2) mod n(S-{n(n+1)\over2})\bmod n(S−2n(n+1)​)modn个数分配给a1,...,a(S−n(n+1)2) mod na_1,...,a_{(S-{n(n+1)\over2})\bmod n}a1​,...,a(S−2n(n+1)​)modn​

写在一起就是

ai=i+⌊S−n(n+1)2n⌋+[i⩽(S−n(n+1)2) mod n], i=1,2,...,na_i=i+\left\lfloor{S-{n(n+1)\over2}\over n}\right\rfloor+\left[i\leqslant\left(S-{n(n+1)\over2}\right)\bmod n\right],~i=1,2,...,n ai​=i+⌊nS−2n(n+1)​​⌋+[i⩽(S−2n(n+1)​)modn], i=1,2,...,n
