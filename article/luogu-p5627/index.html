题解 - [Luogu P5627] 【AFOI-19】sum与prod

发表于 2020-08-12 更新于 2020-11-22 分类于 程序设计 ， 题解

题目链接

原始题面

题目背景

SY终于整理好了她凌乱的被子, 刚来到教室的她就收到了QM传来的一张字条.

To: Dear SY

    ~~~~    你看看我昨晚梦到的式子, 解出来给你糖吃

From: Your QM

SY自然是无法拒绝 C6H12O6C_{6}H_{12}O_{6}C6​H12​O6​ 的诱惑啦, 不过她看到字条背面花里胡哨的式子时傻眼了. . 但是SY还是很想吃糖

题目描述

求

∑i=12nlog⁡2(∏j=1ilowbit(j))\sum_{i=1}^{2^{n}}\log_{2}{(\prod_{j = 1}^{i}lowbit(j))}i=1∑2n​log2​(j=1∏i​lowbit(j))

的结果

其中 lowbit(x)lowbit(x)lowbit(x) 意指x&(~x+1)的结果

输入格式

一行, 一个整数n

输出格式

一行, 一个整数, 为答案模 109+710^9+7109+7 的结果

输入输出样例

输入 #1

2

输出 #1

5

输入 #2

5

输出 #2

447

说明/提示

对于前 20%20\%20% 的数据, 有 ≤n≤60\leq n \leq 60≤n≤60

对于前 50%50\%50% 的数据, 有 ≤n≤104\leq n \leq 10^4≤n≤104

对于前 100%100\%100% 的数据, 有 ≤n≤262\leq n \leq 2^{62}≤n≤262

解题思路

容易得到, 原式即为

+0+0+1+0+1+2+0+1+2+2...+0+1+2+2+...+k+k+...+k⏟k+...+n⏟1\begin{aligned} &+0\\ &+0+1\\ &+0+1+2\\ &+0+1+2+2\\ &...\\ &+0+1+2+2+...+\underbrace{k+k+...+k}_{k}+...+\underbrace n_1\\ \end{aligned}​+0+0+1+0+1+2+0+1+2+2...+0+1+2+2+...+kk+k+...+k​​+...+1n​​​

即 n+(1+2n−1)(2n−n−1)n+(1+2^{n-1})(2^n-n-1)n+(1+2n−1)(2n−n−1)
