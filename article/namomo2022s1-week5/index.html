题目链接

1s, 512MB

题目描述

小红有一个秘密的数组 AAA, 你知道这个数组 AAA 的长度是 NNN. 小红会给你 QQQ 个提示, 第 iii 个提示是告诉你数组 AAA 中 LiL_iLi​ 到 RiR_iRi​ 连续元素的区间和. 问你能否根据小红的这 QQQ 个提示知道数组 AAA 中所有元素的和? 如果能, 输出 Yes; 否则, 输出 No

输入格式

第一行两个整数 NNN 和 qqq, 接下来有 QQQ 行, 每行两个整数 LiL_iLi​ 和 RiR_iRi​

输出格式

Yes 或 No, 表示能不能

样例输出

1≤N≤2×1051\leq N\leq 2 \times 10^51≤N≤2×105

1≤Q≤min⁡(2×105,N(N+1)2)1\leq Q\leq \min(2 \times 10^5,\frac{N(N+1)}{2})1≤Q≤min(2×105,2N(N+1)​)

503 - 选数2 (40th Petrozavodsk Programming Camp, Winter 2021, day 1, M)

题目链接

1s, 1024MB

题目描述

有NNN个数, 小t准备在这NNN个数中选出若干个.满足这些数的最大值 小于等于 这些数的平均值的 kkk 倍

小t想让自己选的数的个数尽可能多, 试求出有多少数字是不可能被小t选到的

我们设MMM为最多能选出的数的个数, 一个数字不可能被选到 当且仅当不出现在任何一个选出MMM个数的方案之中

输入格式

第一行一个正整数NNN 表示数的个数

接下来一行NNN个正整数, 分别表示这NNN个数, 两个数字之间用空格隔开

最后一行两个正整数ppp和qqq, 表示kkk,(k=pqk = \frac{p}{q}k=qp​ 且 k>1k > 1k>1)

输出格式

第一行输出MMM, 表示不可能被选到的数的个数

接下来一行输出MMM个正整数, 分别表示不可能被选到的数字在原序列中的下标, 并按升序排序. 两个数字之间用空格隔开
数据范围

对于所有数据, 满足1≤n≤2⋅1051 \leq n \leq 2 \cdot 10^51≤n≤2⋅105, 0≤ai≤1090 \leq a_i \leq 10^90≤ai​≤109, 1≤q<p≤10001 \leq q < p \leq 10001≤q<p≤1000
