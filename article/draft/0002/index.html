简要讲讲代码的含义

_Generic(a + 0.1f, int : b, float : c, default : b)++;

a为int, a + 0.1f为float, 所以_Generic执行float对应的操作, 即返回c, 最终该语句为c++

_Generic(a += 1.1f, int : b, float : c, default : b)++;

a为int, a += 1.1f不改变a的值, _Generic判断a的类型, 执行int对应的操作, 即返回b, 最终该语句为b++
