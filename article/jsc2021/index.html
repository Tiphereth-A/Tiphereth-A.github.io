A - Competition

原始题面

Time Limit: 2 sec / Memory Limit: 1024 MB

Score: 100 points

Problem Statement

A Supermarket Takahashi sells an XXX-gram beef pack for YYY yen

Another Supermarket Snuke has decided to sell a beef pack at a lower price per gram

In Snuke, one beef pack weighs ZZZ grams. What is the greatest possible price (a non-negative integer) for Snuke’s beef pack such that it is strictly cheaper than Takahashi’s beef pack per gram?

Constraints

  • All values in input are integers
  • 1⩽X,Y,Z⩽1031\leqslant X,Y,Z\leqslant 10^31⩽X,Y,Z⩽103

Input

Input is given from Standard Input in the following format:

X Y ZX\ Y\ ZX Y Z

Output

Print the answer

Sample Input 1

100 200 100

Sample Output 1

199

Both stores sell 100100100-gram packs, so Snuke can just make it one yen cheaper than that in Takahashi

Sample Input 2

103 971 593

Sample Output 2

5590

Takahashi sells beef for 971103=9.4271...{971\over 103}=9.4271...103971​=9.4271...
yen per gram. Snuke can sell 593593593 grams of beef for 559055905590 yen to make it 5590593=9.4266...{5590\over 593}=9.4266...5935590​=9.4266... yen per gram

Sample Input 3

1000 1 1

Sample Output 3

0

The price is allowed to be 000 yen

解题思路

⌊yzx⌋−[x∣yz]\lfloor{yz\over x}\rfloor-[x\mid yz]⌊xyz​⌋−[x∣yz]

原始题面

Time Limit: 2 sec / Memory Limit: 1024 MB

Score: 400 points

Problem Statement

You are given a prime number PPP not less than 222, which you don’t like

Let’s call an array of integers A1,A2,...,ANA_1,A_2,...,A_NA1​,A2​,...,AN​ very good if it satisfies the following condition:

  • there is no iii with 1⩽i⩽N1\leqslant i\leqslant N1⩽i⩽N and A1+A2+...+Ai≡0(modP)A_1+A_2+...+A_i\equiv0\pmod PA1​+A2​+...+Ai​≡0(modP)

Consider all (P−1)N(P-1)^N(P−1)N arrays of length NNN with elements from 111 to P−1P-1P−1. How many of them are very good?

As this number can be very big, output it modulo 109+710^9+7109+7

  • 1⩽N⩽1091\leqslant N\leqslant 10^91⩽N⩽109
  • 2⩽P⩽1092\leqslant P\leqslant 10^92⩽P⩽109

Print the count modulo 109+710^9+7109+7

Two arrays, (1,1,2)(1,1,2)(1,1,2) and (2,2,1)(2,2,1)(2,2,1), satisfy the condition

Sample Input 2

Sample Output 2

题意简述

给定质数 PPP, 求有多少序列 (A1,A2,...,AN)(\def\enum#1{A_{ #1}}\enum{1},\enum{2},...,\enum{N})(A1​,A2​,...,AN​) 满足:

∀i∈[1,n]N, ∑j=1iAj≡/ 0(modP)\forall i\in[1,n]_{\N},~\sum_{j=1}^iA_j{\equiv}\llap{/\,}0\pmod P∀i∈[1,n]N​, j=1∑i​Aj​≡/​0(modP)

解题思路

显然, 当 n=1n=1n=1 时答案为 p−1p-1p−1, 对应合法序列即为 (1),(2),...,(p−1)\def\enum#1{({ #1})}\enum{1},\enum{2},...,\enum{p-1}(1),(2),...,(p−1)

之后在这些合法序列后插入新数时, 每个序列都有且仅有一个数使得这个数插入后该序列非法 (该数即为 (−∑iai) mod p(-\sum_{i}a_i)\bmod p(−∑i​ai​)modp)

故答案为 (p−1)(p−2)n−1(p-1)(p-2)^{n-1}(p−1)(p−2)n−1

E - Level K Palindrome

原始题面

Time Limit: 2 sec / Memory Limit: 1024 MB

Score: 500 points

Problem Statement

As a token of his gratitude, Takahashi has decided to give Snuke a level-KKK palindrome. A level-LLL palindrome, where LLL is a non-negative integer, is defined as follows:

  • Let rev⁡(s)\operatorname{rev}(s)rev(s) denote the reversal of a string sss
  • A string sss is said to be a palindrome when s=rev⁡(s)s=\operatorname{rev}(s)s=rev(s)
  • The empty string and a string that is not a palindrome are level-000 palindromes
  • For any non-empty level-(L−1L-1L−1) palindrome ttt, the concatenation of ttt,rev⁡(t)\operatorname{rev}(t)rev(t) in this order is a level-LLL palindrome
  • For any level-(L−1L-1L−1) palindrome ttt and any character ccc, the concatenation of ttt,ccc,rev⁡(t)\operatorname{rev}(t)rev(t) in this order is a level-LLL palindrome

Now, Takahashi has a string SSS. Determine whether it is possible to make SSS an exactly level-KKK palindrome by doing the following action zero or more times: choose a character in SSS and change it to another lowercase English letter. If it is possible, find the minimum number of changes needed to make SSS a level-KKK palindrome

Constraints

  • KKK is an integer
  • 0⩽K⩽5×1050\leqslant K\leqslant 5\times 10^50⩽K⩽5×105
  • SSS consists of lowercase English letters
  • 1⩽∣S∣⩽5×1051\leqslant |S|\leqslant 5\times 10^51⩽∣S∣⩽5×105

Input

Input is given from Standard Input in the following format:

K SK\ SK S

Output

If it is possible to get an exactly level-KKK palindrome, print the minimum number of changes needed. If it is impossible, print impossible

Sample Input 1

4
aabaaaabaa

Sample Output 1

0

We can find the level of aabaaaabaa as follows:

  • the empty string is a level-000 palindrome
  • a is a concatenation of (empty string), a, (empty string) in this order, so it is a level-111 palindrome
  • aa is a concatenation of a, a in this order, so it is a level-222 palindrome
  • aabaa is a concatenation of aa, b, aa in this order, so it is a level-333 palindrome
  • aabaaaabaa is a concatenation of aabaa, aabaa in this order, so it is a level-444 palindrome

Thus, aabaaaabaa is already a level-444 palindrome and needs no changes

Sample Input 2

2
aabaaaabaa

Sample Output 2

4

We can, for example, change aabaaaabaa to acbcaacbca to get a level-222 palindrome

Note that aabaaaabaa is not a level-222 palindrome

Sample Input 3

3
aabaaaabaa

Sample Output 3

impossible

Sample Input 4

5
aabaaaabaa

Sample Output 4

impossible

Sample Input 5

2
acaabcbababaaac

Sample Output 5

6

题意简述

本题所有的字符串均指只由小写英文字母构成的字符串

对字符串 sss,

  • 定义其反转为: rev⁡(s)\operatorname{rev}(s)rev(s), 则 sss 是回文串  ⟺  s=rev⁡(s)\iff s=\operatorname{rev}(s)⟺s=rev(s)
  • +++ 运算定义为字符串的拼接
  • 定义字符串上的变换为: 将其中某一字符替换为一小写英文字母

定义 kkk 阶回文串如下:

  • 空串, 非回文串为 000 阶回文串
  • 对 iii 阶非空回文串 sss, 定义 s+rev⁡(s)s+\operatorname{rev}(s)s+rev(s) 为 i+1i+1i+1 阶回文串
  • 对 iii 阶回文串 sss 和单个字符 ccc, 定义 s+c+rev⁡(s)s+c+\operatorname{rev}(s)s+c+rev(s) 为 i+1i+1i+1 阶回文串

给一字符串 sss, 问至少经几次变换可使其恰好为 kkk 阶回文串

解题思路

显然, 若有解则 kkk 不可能过大
