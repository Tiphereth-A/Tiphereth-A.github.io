C - Dice and Coin

原始题面

Problem Statement

Snuke has a fair NNN-sided die that shows the integers from 111 to NNN with equal probability and a fair coin. He will play the following game with them:

  1. Throw the die. The current score is the result of the die.
  2. As long as the score is between 111 and K−1K-1K−1 (inclusive), keep flipping the coin. The score is doubled each time the coin lands heads up, and the score becomes 000 if the coin lands tails up.
  3. The game ends when the score becomes 000 or becomes KKK or above. Snuke wins if the score is K or above, and loses if the score is 000.

You are given NNN and KKK. Find the probability that Snuke wins the game.

Constraints

  • 1≤N≤1051≤N≤10^51≤N≤105
  • 1≤K≤1051≤K≤10^51≤K≤105
  • All values in input are integers.

Input

Input is given from Standard Input in the following format:

N KN\ KN K

Output

Print the probability that Snuke wins the game. The output is considered correct when the absolute or relative error is at most 10−910^{-9}10−9.

Sample Input 1

Sample Output 1

  • If the die shows 111, Snuke needs to get four consecutive heads from four coin flips to obtain a score of 10 or above. The probability of this happening is 13×(12)4=148{1\over3}×({1\over2})^4={1\over48}31​×(21​)4=481​.
  • If the die shows 222, Snuke needs to get three consecutive heads from three coin flips to obtain a score of 10 or above. The probability of this happening is 13×(12)3=124{1\over3}×({1\over2})^3={1\over24}31​×(21​)3=241​.
  • If the die shows 333, Snuke needs to get two consecutive heads from two coin flips to obtain a score of 10 or above. The probability of this happening is 13×(12)2=112{1\over3}×({1\over2})^2={1\over12}31​×(21​)2=121​.

Thus, the probability that Snuke wins is 148+124+112=748≃0.1458333333{1\over48}+{1\over24}+{1\over12}={7\over48}≃0.1458333333481​+241​+121​=487​≃0.1458333333.

Your objective is to paint each vertex in the tree white or black (it is fine to paint all vertices the same color) so that the following condition is satisfied:

  • For any two vertices painted in the same color, the distance between them is an even number.

Find a coloring of the vertices that satisfies the condition and print it. It can be proved that at least one such coloring exists under the constraints of this problem.

Print a coloring of the vertices that satisfies the condition, in NNN lines. The iii-th line should contain 000 if Vertex iii is painted white and 111 if it is painted black.

F - XOR Matching

原始题面

Problem Statement

Construct a sequence a={a1,a2,...,a2M+1}a= \{a_1, a_2, ..., a_{2^{M+1}}\}a={a1​,a2​,...,a2M+1​} of length 2M+12^{M+1}2M+1 that satisfies the following conditions, if such a sequence exists.

  • Each integer between 000 and 2M−12^M-12M−1 (inclusive) occurs twice in aaa.
  • For any iii and jjj (i<ji<ji<j) such that ai=aja_i=a_jai​=aj​, the formula ai xor ai+1 xor ... xor aj=Ka_i\ xor\ a_{i+1}\ xor\ ...\ xor\ a_j=Kai​ xor ai+1​ xor ... xor aj​=K holds.
What is xor (bitwise exclusive or)?The xor of integers $c_1,c_2,...,c_n$ is defined as follows:
  • When c1 xor c2 xor ... xor cnc_1\ xor\ c_2\ xor\ ...\ xor\ c_nc1​ xor c2​ xor ... xor cn​ is written in base two, the digit in the 2k2^k2k's place (k≥0k≥0k≥0) is 1 if the number of integers among c1,c2,...cmc_1,c_2,...c_mc1​,c2​,...cm​ whose binary representations have 111 in the 2k2^k2k's place is odd, and 000 if that count is even.

For example, 3 xor 5=63\ xor\ 5=63 xor 5=6. (If we write it in base two: 011 xorxorxor 101 = 110.)

Constraints

  • All values in input are integers.
  • 0≤M≤170≤M≤170≤M≤17
  • 0≤K≤1090≤K≤10^90≤K≤109

Input

Input is given from Standard Input in the following format:

M KM\ KM K

Output

If there is no sequence aaa that satisfies the condition, print -1.

If there exists such a sequence aaa, print the elements of one such sequence aaa with spaces in between.

If there are multiple sequences that satisfies the condition, any of them will be accepted.

Sample Input 1

Sample Output 1

For this case, there are multiple sequences that satisfy the condition.

For example, when a={0,0,1,1}a = \{0,0,1,1\}a={0,0,1,1}, there are two pairs (i,j)(i<j)(i, j) (i<j)(i,j)(i<j) such that ai=aja_i=a_jai​=aj​: (1,2)(1,2)(1,2) and (3,4)(3,4)(3,4). Since a1 xor a2=0a_1\ xor\ a_2=0a1​ xor a2​=0 and a3 xor a4=0a_3\ xor\ a_4=0a3​ xor a4​=0, this sequence aaa satisfies the condition.

解题思路

首先我们注意到

∀k∈N+,⨁i=02k−1i=0\forall k\in\N^+,\bigoplus_{i=0}^{2^k-1}i=0∀k∈N+,i=0⨁2k−1​i=0

用数学归纳法证明即可, 或者考虑 Sierpinski 三角形

其次我们知道异或的逆运算是异或, 所以我们可以这样构造:

  • 若K⩾2MK\geqslant 2^MK⩾2M, 则一定无解

  • 若M=1M=1M=1, 则K=0K=0K=0有解, K=1K=1K=1无解

  • 其余情况我们可以这样构造

    0 1 ... K-1 K+1 ... 2^M-2 2^M-1 K 2^M-1 2^M-2 ... K+1 K-1 ... 1 0 K
