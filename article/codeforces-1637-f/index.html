题解 - [CodeForces 1637 F] Towers

发表于 2022-03-02 分类于 程序设计 ， 题解

题目链接

原始题面

time limit per test: 2 seconds

memory limit per test: 256 megabytes

input: standard input

output: standard output

You are given a tree with nnn vertices numbered from 111 to nnn. The height of the iii-th vertex is hih_ihi​. You can place any number of towers into vertices, for each tower you can choose which vertex to put it in, as well as choose its efficiency. Setting up a tower with efficiency eee costs eee coins, where e>0e > 0e>0

It is considered that a vertex xxx gets a signal if for some pair of towers at the vertices uuu and vvv (u≠vu \neq vu​=v, but it is allowed that x=ux = ux=u or x=vx = vx=v) with efficiencies eue_ueu​ and eve_vev​, respectively, it is satisfied that min⁡(eu,ev)≥hx\min(e_u, e_v) \geq h_xmin(eu​,ev​)≥hx​ and xxx lies on the path between uuu and vvv

Find the minimum number of coins required to set up towers so that you can get a signal at all vertices

Input

The first line contains an integer nnn (2≤n≤200 0002 \le n \le 200\,0002≤n≤200000) — the number of vertices in the tree

The second line contains nnn integers hih_ihi​ (1≤hi≤1091 \le h_i \le 10^91≤hi​≤109) — the heights of the vertices

Each of the next n−1n- 1n−1 lines contain a pair of numbers vi,uiv_i, u_ivi​,ui​ (1≤vi,ui≤n1 \le v_i, u_i \le n1≤vi​,ui​≤n) — an edge of the tree. It is guaranteed that the given edges form a tree

Output

Print one integer — the minimum required number of coins

Note

In the first test case it’s optimal to install two towers with efficiencies 222 at vertices 111 and 333

In the third test case it’s optimal to install two towers with efficiencies 666 at vertices 111 and 222

题意简述

在一棵无根树上放若干个塔, 每个塔有权值 eee, 树上结点自带权值 hhh, 要求树上任意结点均在某两个塔之间的简单路径上, 且这两个塔的权值均不小于该结点的权值, 求塔权值的最小和

解题思路

显然塔应都放在叶子结点上

为方便起见, 我们让权值最大的点为根节点, 令其为 rrr, 此时我们只需让叶子结点里有两个塔的权值为 hrh_rhr​ 即可

我们考虑一条从根节点到叶子结点的链, 其中根结点的一端相当于有了一个权值为 hrh_rhr​ 的塔, 那么另一端只需放一个权值为该链上的结点权值次大值的塔即可

这有 2500 ?
