题解 - [POJ 3361] Gaussian Prime Factors

发表于 2020-07-27 更新于 2020-12-06 分类于 程序设计 ， 题解

题目链接

原始题面

Description

Let a,b,c,da, b, c, da,b,c,d be integers. The complex number a+bja+bja+bj, where j2=−1j^2 = -1j2=−1, is a factor of c+djc+djc+dj, if there exist integers eee and fff such that

c+dj=(a+bj)(e+fj)c + dj = (a + bj)(e + fj)c+dj=(a+bj)(e+fj)

A complex number a+bja + bja+bj where aaa and bbb are integers is a Gaussian prime if the factors are 1,−1,−a−bj1, -1, -a - bj1,−1,−a−bj and a+bja + bja+bj only

The following are Gaussian primes: 1+j,1−j,1+2j,1−2j,31 + j, 1 - j, 1 + 2j, 1 - 2j, 31+j,1−j,1+2j,1−2j,3 and 777

The Gaussian prime factors of 5 are:

1+2j1 + 2j1+2j and 1−2j1 - 2j1−2j, or
2+j2 + j2+j and 2−j2 - j2−j, or
−1−2j-1 - 2j−1−2j and −1+2j-1 + 2j−1+2j, or
−2−j-2 - j−2−j and −2+j-2 + j−2+j

Write a program that finds all the Gaussian prime factors of a positive integer

Input

One line of input per case. The line represents a positive integer nnn

Output

One line of output per test case. The line represents the Gaussian prime factors of nnn. If a+bja + bja+bj is a Gaussian prime factor of nnn, then a>0a > 0a>0, ∣b∣≥a|b| ≥ a∣b∣≥a, if b≠0b ≠ 0b​=0. If b=0b = 0b=0, the output must be aaa

Sample Input

2
5
6
700

Sample Output

Case #1: 1+j, 1-j 
Case #2: 1+2j, 1-2j 
Case #3: 1+j, 1-j, 3 
Case #4: 1+j, 1-j, 1+2j, 1-2j, 7

Hint

Output the Gaussian prime factors in ascending order of aaa. If there are more than one factors with the same aaa, output them in ascending order of bbb by absolute value. If two conjugate factors coexist, the one with a positive imaginary part precedes that with a negative imaginary part

Source

Manila 2006

题意简述

给出整数sss, 输出在Z[j]\Z[j]Z[j]下的所有非平凡素因子, 其中j2=−1j^2=-1j2=−1

解题思路

题面的Hint里提到了做法, 这里简要概括一下

  • 首先对sss做质因数分解
  • 其次考察sss的每个质因子ppp
    • 若p≡3(mod4)p\equiv 3\pmod4p≡3(mod4), 说明ppp不能表示成二平方和的形式, 故ppp也是Z[j]\Z[j]Z[j]下的素数(即Gauss素数)
    • 若p≡1(mod4)p\equiv 1\pmod4p≡1(mod4), 说明ppp能表示成二平方和的形式, 设p=a2+b2p=a^2+b^2p=a2+b2
      则a+bi, a−bia+bi,~a-bia+bi, a−bi是Gauss素数而ppp不是

最后输出结果的时候要按aaa升序输出, 如果对于相同的aaa, 有多个对应的b≠0b\ne0b​=0, 则按bbb升序输出一组共轭的Gauss素数
