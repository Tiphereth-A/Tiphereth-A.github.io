题解 - [POJ 3734] Gaussian Prime Factors

发表于 2021-07-12 更新于 2021-10-30 分类于 程序设计 ， 题解

题目链接

原始题面

Description

Panda has received an assignment of painting a line of blocks. Since Panda is such an intelligent boy, he starts to think of a math problem of painting. Suppose there are N blocks in a line and each block can be paint red, blue, green or yellow. For some myterious reasons, Panda want both the number of red blocks and green blocks to be even numbers. Under such conditions, Panda wants to know the number of different ways to paint these blocks.

Input

The first line of the input contains an integer T(1≤T≤100), the number of test cases. Each of the next T lines contains an integer N(1≤N≤10^9) indicating the number of blocks.

Output

For each test cases, output the number of ways to paint the blocks in a single line. Since the answer may be quite large, you have to module it by 10007.

Sample Input

2
1
2

Sample Output

2
6

Source

PKU Campus 2009 (POJ Monthly Contest – 2009.05.17), Simon

题意简述

给 nnn 个方块排成一排, 每个方块都要染上红绿蓝黄四种颜色之一, 求红色方块和绿色方块个数均为偶数的染色方案数

解题思路

法一 (容斥 + 二项式/多项式定理)

显然答案应为

无限制的方案数 - 红色方块为奇数的方案数 - 绿色方块为奇数的方案数 + 红色和绿色方块均为奇数的方案数

即

4n−2A+B(1)4^n-2A+B\tag{1}4n−2A+B(1)

其中

  • A=∑i=1⌈n2⌉(n2i−1)3n−2i+1(2)A=\sum_{i=1}^{\lceil{n\over2}\rceil} {n\choose 2i-1}3^{n-2i+1}\tag{2}A=i=1∑⌈2n​⌉​(2i−1n​)3n−2i+1(2)

  • B=∑i=1⌈n2⌉∑j=1⌈n2⌉n!(2i−1)!(2j−1)!(n−2i−2j+2)!2n−2i−2j+2(3)B=\sum_{i=1}^{\lceil{n\over2}\rceil}\sum_{j=1}^{\lceil{n\over2}\rceil} {n!\over (2i-1)!(2j-1)!(n-2i-2j+2)!}2^{n-2i-2j+2}\tag{3}B=i=1∑⌈2n​⌉​j=1∑⌈2n​⌉​(2i−1)!(2j−1)!(n−2i−2j+2)!n!​2n−2i−2j+2(3)

由二项式定理和多项式定理, 有

  • A=(3+1)n−(3−1)n2(4)A={(3+1)^n-(3-1)^n\over2}\tag{4}A=2(3+1)n−(3−1)n​(4)

  • B=(2+1+1)n−(2+1−1)n−(2−1+1)n+(2−1−1)n4(5)B={(2+1+1)^n-(2+1-1)^n-(2-1+1)^n+(2-1-1)^n\over 4}\tag{5}B=4(2+1+1)n−(2+1−1)n−(2−1+1)n+(2−1−1)n​(5)

将 (4)\text{(4)}(4), (5)\text{(5)}(5) 代入 (1)\text{(1)}(1), 则答案为

4n−1+2n−14^{n-1}+2^{n-1}4n−1+2n−1

4n+2n+14=4n−1+2n−1{4^n+2^{n+1}\over4}=4^{n-1}+2^{n-1}44n+2n+1​=4n−1+2n−1
