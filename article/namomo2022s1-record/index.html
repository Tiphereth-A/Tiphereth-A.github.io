103 - 子串的最大差 (CF817D)

题目链接

定义序列的最大差为序列中最大数与最小数的差. 比如 (3,1,4,5,6)(3,1,4,5,6)(3,1,4,5,6) 的最大差为 6−1=56 - 1 = 56−1=5, (2,2)(2,2)(2,2) 的最大差为 2−2=02 - 2 = 02−2=0

定义一个序列的子串为该序列中连续的一段序列

给定一个长度为 nnn 的数组 a1,a2,…,ana_1,a_2,\dots ,a_na1​,a2​,…,an​, 请求出这个序列的所有子串的最大差之和

输入格式

第一行一个数字 nnn

接下来一行 nnn 个整数 a1,a2,…,ana_1, a_2, \dots, a_na1​,a2​,…,an​

输出格式

一个数, 表示答案

样例输出

所有数据保证 1≤n≤500000,0≤ai≤1081\leq n\leq 500000, 0 \leq a_i \leq 10^81≤n≤500000,0≤ai​≤108

解题思路

DP + 单调栈

即求

∑i=1n∑j=1i(max⁡x∈i..j{ax}−min⁡x∈i..j{ax})\sum_{i=1}^n\sum_{j=1}^i\left(\max_{x\in i..j}\{a_x\}-\min_{x\in i..j}\{a_x\}\right)i=1∑n​j=1∑i​(x∈i..jmax​{ax​}−x∈i..jmin​{ax​})

令

  • Li=max⁡({0}∪{x<i∣ax>ai})L_i=\max(\{0\}\cup\{x<i\mid a_x>a_i\})Li​=max({0}∪{x<i∣ax​>ai​})

  • Ri=min⁡({n+1}∪{x>i∣ax>ai})R_i=\min(\{n+1\}\cup\{x>i\mid a_x>a_i\})Ri​=min({n+1}∪{x>i∣ax​>ai​})

  • li=max⁡({0}∪{x<i∣ax<ai})l_i=\max(\{0\}\cup\{x<i\mid a_x<a_i\})li​=max({0}∪{x<i∣ax​<ai​})

  • ri=min⁡({n+1}∪{x>i∣ax<ai})r_i=\min(\{n+1\}\cup\{x>i\mid a_x<a_i\})ri​=min({n+1}∪{x>i∣ax​<ai​})

则

∑i=1n∑j=1i(max⁡x∈i..j{ax}−min⁡x∈i..j{ax})=∑i=1n∑j=1imax⁡x∈i..j{ax}−∑i=1n∑j=1imin⁡x∈i..j{ax}=∑i=1nai(Ri−i)(i−Li)−∑i=1nai(ri−i)(i−li)\begin{aligned} \sum_{i=1}^n\sum_{j=1}^i\left(\max_{x\in i..j}\{a_x\}-\min_{x\in i..j}\{a_x\}\right)&=\sum_{i=1}^n\sum_{j=1}^i\max_{x\in i..j}\{a_x\}-\sum_{i=1}^n\sum_{j=1}^i\min_{x\in i..j}\{a_x\}\\ &=\sum_{i=1}^n a_i(R_i-i)(i-L_i)-\sum_{i=1}^n a_i(r_i-i)(i-l_i) \end{aligned}i=1∑n​j=1∑i​(x∈i..jmax​{ax​}−x∈i..jmin​{ax​})​=i=1∑n​j=1∑i​x∈i..jmax​{ax​}−i=1∑n​j=1∑i​x∈i..jmin​{ax​}=i=1∑n​ai​(Ri​−i)(i−Li​)−i=1∑n​ai​(ri​−i)(i−li​)​

然后用单调栈把 Li,Ri,li,riL_i,R_i,l_i,r_iLi​,Ri​,li​,ri​ 求出来即可

104 - no crossing (CF793D)

题目链接

给出一个有向图, 找一条恰好经过 kkk 个点的最短路径, 要求每次选的边不能跃过之前已经经过的节点. 即对于路径中的边 x→yx \rightarrow yx→y, 不存在以前经过的点 ttt 使得三者的编号满足 min⁡(x,y)≤t≤max⁡(x,y)\min(x,y) \leq t \leq \max(x,y)min(x,y)≤t≤max(x,y)

输入格式

第一行三个数字 n,k,mn,k,mn,k,m

接下来mmm行, 每行 333 个整数 ai,bi,cia_i,b_i,c_iai​,bi​,ci​ 表示存在一条从 ai→bia_i \rightarrow b_iai​→bi​, 长度为 cic_ici​ 的有向边

输出格式

一个数, 表示答案. 如果不存在任何一条路径满足条件, 则输出 −1-1−1

样例1输出

数据规模

所有数据保证 1≤n,k≤100,0≤m≤2000,1≤ai,bi≤n,1≤ci≤10001\leq n,k\leq 100, 0 \leq m \leq 2000 , 1 \leq a_i,b_i \leq n, 1 \leq c_i \leq 10001≤n,k≤100,0≤m≤2000,1≤ai​,bi​≤n,1≤ci​≤1000

Note

样例一的最短路径为 1→6→2→41 \rightarrow 6 \rightarrow 2 \rightarrow 41→6→2→4. 1→6→2→71 \rightarrow 6 \rightarrow 2 \rightarrow 71→6→2→7 是不正确的, 因为2<6<72 < 6 < 72<6<7

解题思路

区间 DP + 滚动数组

如果可以跃过之前经过的结点, 那直接对邻接矩阵求个快速幂就可以了

在有了上述约束后, 我们不难发现随着走的步数越来越多, 可以走的范围是越来越小的

所以我们这样设计状态方程

令

  • f(k,u,v)f(k,u,v)f(k,u,v) 表示走了 kkk 步时, 可以走的范围为 (min⁡{u,v},max⁡{u,v})(\min\{u,v\},\max\{u,v\})(min{u,v},max{u,v}), 目前在 vvv 处的最短路
  • d(u,v)d(u,v)d(u,v) 为 uuu 到 vvv 的边权, 若两点不相邻则为 ∞\infty∞

则

f(k,u,v)={min⁡x>v{f(k−1,u,x)+d(x,v),f(k−1,x,u)+d(u,v)},u<vmin⁡x<v{f(k−1,u,x)+d(x,v),f(k−1,x,u)+d(u,v)},u>vf(k,u,v)=\begin{cases} \min_{x>v}\{f(k-1,u,x)+d(x,v),f(k-1,x,u)+d(u,v)\},&u<v\\ \min_{x<v}\{f(k-1,u,x)+d(x,v),f(k-1,x,u)+d(u,v)\},&u>v\\ \end{cases}f(k,u,v)={minx>v​{f(k−1,u,x)+d(x,v),f(k−1,x,u)+d(u,v)},minx<v​{f(k−1,u,x)+d(x,v),f(k−1,x,u)+d(u,v)},​u<vu>v​

显然可以用滚动数组把 kkk 这一维优化一下

复杂度

O(n3k)O(n^3k)O(n3k)
