题解 - [AtCoder ARC110D] Binomial Coefficient is Fun

发表于 2020-12-06 分类于 程序设计 ， 题解

题目链接

原始题面

Time Limit: 2 sec / Memory Limit: 1024 MB

Score : 600 points

Problem Statement

We have a sequence AAA of NNN non-negative integers

Compute the sum of ∏i=1N(BiAi)\prod_{i=1}^N{B_i\choose A_i}∏i=1N​(Ai​Bi​​)
over all sequences BBB of NNN non-negative integers whose sum is at most MMM, and print it modulo (109+710^9+7109+7)

Here, (BiAi){B_i\choose A_i}(Ai​Bi​​)
, the binomial coefficient, denotes the number of ways to choose AiA_iAi​ objects from BiB_iBi​ objects, and is 0 when Bi<AiB_i<A_iBi​<Ai​

Constraints

  • All values in input are integers
  • 1≤N≤20001≤N≤20001≤N≤2000
  • 1≤M≤1091≤M≤10^91≤M≤109
  • 0≤Ai≤20000≤A_i≤20000≤Ai​≤2000

Input

Input is given from Standard Input in the following format:

N MA1 A2 ... ANN\ M\\A_1\ A_2\ ...\ A_NN MA1​ A2​ ... AN​

Output

Print the sum of ∏i=1N(BiAi)\prod_{i=1}^N{B_i\choose A_i}∏i=1N​(Ai​Bi​​), modulo (109+710^9+7109+7)

Sample Input 1

3 5
1 2 1

Sample Output 1

8

There are four sequences B
such that ∏i=1N(BiAi)\prod_{i=1}^N{B_i\choose A_i}∏i=1N​(Ai​Bi​​) is at least 1:

  • B={1,2,1}B=\{1,2,1\}B={1,2,1}, where ∏i=1N(BiAi)=(11)×(22)×(11)=1\prod_{i=1}^N{B_i\choose A_i}={1\choose1}×{2\choose2}×{1\choose1}=1∏i=1N​(Ai​Bi​​)=(11​)×(22​)×(11​)=1
  • B={2,2,1}B=\{2,2,1\}B={2,2,1}, where ∏i=1N(BiAi)=(21)×(22)×(11)=2\prod_{i=1}^N{B_i\choose A_i}={2\choose1}×{2\choose2}×{1\choose1}=2∏i=1N​(Ai​Bi​​)=(12​)×(22​)×(11​)=2
  • B={1,3,1}B=\{1,3,1\}B={1,3,1}, where ∏i=1N(BiAi)=(11)×(32)×(11)=3\prod_{i=1}^N{B_i\choose A_i}={1\choose1}×{3\choose2}×{1\choose1}=3∏i=1N​(Ai​Bi​​)=(11​)×(23​)×(11​)=3
  • B={1,2,2}B=\{1,2,2\}B={1,2,2}, where ∏i=1N(BiAi)=(11)×(22)×(21)=2\prod_{i=1}^N{B_i\choose A_i}={1\choose1}×{2\choose2}×{2\choose1}=2∏i=1N​(Ai​Bi​​)=(11​)×(22​)×(12​)=2

The sum of these is 1+2+3+2=81+2+3+2=81+2+3+2=8

题意简述

给定正整数n,mn,mn,m和a1,a2,...,an\def\enum#1{a_{ #1}}\enum{1},\enum{2},...,\enum{n}a1​,a2​,...,an​, 求

∑b1,b2,...,bn>0∑i=1nbi⩽m∏i=1n(biai)\sum_{\begin{smallmatrix} b_1,b_2,...,b_n>0\\\\ \sum_{i=1}^nb_i\leqslant m \end{smallmatrix}}\prod_{i=1}^n{b_i\choose a_i}b1​,b2​,...,bn​>0∑i=1n​bi​⩽m​∑​i=1∏n​(ai​bi​​)

解题思路

不难发现该问题等价于如下问题

给出mmm个相同的小球和nnn个盒子, 盒子编号为1,2,...,n1,2,...,n1,2,...,n, 将小球放入盒子中, 要求第iii个盒子至少放aia_iai​个球, 接下来给每个盒子中的小球标号, 并分别在第iii个盒子里取出aia_iai​种, 问一共有多少种取法

先取出∑i=1nai\sum_{i=1}^na_i∑i=1n​ai​个球放进盒子里, 此时问题可转化为

将m−∑i=1naim-\sum_{i=1}^na_im−∑i=1n​ai​个相同的球放入n+∑i=1nain+\sum_{i=1}^na_in+∑i=1n​ai​个有编号盒子(允许盒子为空)的方案数 (考虑插空法易得)

所以最终答案即为

(m+nm−∑i=1nai)=(m+n∑i=1nai+n){m+n\choose m-\sum_{i=1}^na_i}={m+n\choose\sum_{i=1}^na_i+n}(m−∑i=1n​ai​m+n​)=(∑i=1n​ai​+nm+n​)
